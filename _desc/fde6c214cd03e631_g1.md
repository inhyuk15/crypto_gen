## **2.2 Parameters** 

**Members of** Romulus **.** Romulus consists of Romulus-N, Romulus-M, Romulus-T and RomulusH. Romulus-N implements nonce-based AE (NAE) secure against Nonce-respecting adversaries, and Romulus-M implements nonce Misuse-resistant AE (MRAE) introduced by Rogaway and Shrimpton [69]. Romulus-H implements a Hash function with standard security properties, such as collision resistance and (2nd) preimage resistance. The name Romulus stands for the set of these members. The primary AEAD member of our submission is Romulus-N, and Romulus-H is our only hash function member.
